\block{Abstract}{
As Generative AI grows in popularity, the effortless creation of hyper-realistic manipulated media ('deepfakes') presents a massive challenge to digital authenticity. Historically, researchers relied on massive neural networks to spot spatial inconsistencies directly within raw RGB images. 

In this study, I explored a fundamentally different approach. Instead of analyzing visual semantics, I investigated the digital fingerprint left behind by image compression. I used \textbf{Error Level Analysis (ELA)} to reveal hidden compression artifacts. By processing 10,000 frames from FaceForensics++ into ELA heatmaps and fine-tuning an EfficientNet-B0 architecture with differential learning rates, my optimized model achieved a validation accuracy of \textbf{86.40\%} and an F1-score of \textbf{86.11\%}.
}

\block{Introduction}{
\begin{itemize}
    \item \textbf{The RGB Problem:} Most contemporary detection pipelines feed raw RGB pixels into massive CNNs or Vision Transformers. However, as generative models like diffusion models perfect their visual blending, semantic ``tells'' (like weird lighting) are vanishing.
    \item \textbf{The Forensic Angle:} When an image is spliced, the newly inserted region rarely shares the same JPEG compression history as the original host image. This discrepancy acts as a latent smoking gun.
    \item \textbf{My Approach:} I bridged classic digital forensics with deep learning. By feeding ELA heatmaps into a lightweight EfficientNet-B0 backbone, I forced the network to become an expert at identifying compression anomalies rather than easily-faked visual semantics.
\end{itemize}
}

\block{Methodology \& Architecture}{
\textbf{1. Data Curation}\\
I relied on a carefully selected, perfectly balanced subset of the FaceForensics++ dataset (5,000 pristine, 5,000 manipulated deepfakes). I passed these frames through the ELA pipeline shown to the left to strip away natural visual content, leaving behind a stark heatmap of the underlying compression grid.

\vspace{1cm}
\textbf{2. Network Selection: EfficientNet-B0}\\
To ensure the pipeline remained lightweight for practical deployment, I chose the highly parameter-efficient EfficientNet-B0. I removed the original ImageNet head and built a custom top-level module featuring a global average pooling layer and a single linear node for binary (Real vs. Fake) classification using Binary Cross-Entropy.

\vspace{1cm}
\textbf{3. The Domain Shift Challenge}\\
My biggest hurdle was navigating the severe domain shift between standard photos and ELA heatmaps. A standard transfer learning approach (freezing the pre-trained base) completely failed because the ImageNet-trained filters were unequipped to understand high-frequency, grid-like ELA noise.
}

\block{Training Strategy}{
To overcome the domain shift, I pivoted to full network fine-tuning paired with a \textbf{differential learning rate strategy}. I completely unfroze the base network but protected its generalized edge-detection skills using a very small learning rate, while training the new head aggressively:

\vspace{0.5cm}
\begin{itemize}
    \item \textbf{Base Network $\alpha_{base}$:} $5 \times 10^{-5}$
    \item \textbf{Classifier Head $\alpha_{head}$:} $1 \times 10^{-3}$
    \item \textbf{Optimizer:} Adam with Cosine Annealing scheduler.
    \item \textbf{Augmentation:} Random rotations (15$^\circ$) and horizontal flips to prevent spatial overfitting on the noise grids.
\end{itemize}
}

\textbf{Key Findings:}\\
\begin{itemize}
    \item \textbf{Failure of Frozen Weights:} The baseline run yielded an F1-score of roughly 58.37\%, which is barely better than a random coin toss. Standard transfer learning falls short when migrating to forensic representations.
    \item \textbf{Success of Fine-Tuning:} The impact of unfreezing the lower layers was immediate. The final model achieved \textbf{86.40\% Accuracy}.
    \item \textbf{High Precision:} The model achieved a notably high precision score of \textbf{88.00\%}. This indicates a strong resistance to false positives, which is critical for maintaining trust in a detection system.
\end{itemize}
}

\block{Conclusion}{
By moving away from the computationally expensive trend of massive RGB semantic analysis, I showed that examining the underlying digital physics of an image is an incredibly viable alternative. 

Isolating JPEG compression inconsistencies through ELA forces neural networks to act as digital forensic analysts. While teaching a modern CNN to interpret abstract data required navigating severe domain shifts via differential learning rates, my final results validate the hypothesis: we can build highly accurate, lightweight detectors based purely on compression artifacts.
}
